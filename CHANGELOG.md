## [2.8.1] - 2025-02-12

### 核心亮点

本版本主要完成 **文档完善** 与 **项目结构优化**，重构 README 文档结构，新增详细的 MCP 使用指南和部署配置文档，优化项目展示效果。

### Added

- **文档重构**
  - 拆分 README 文档，新增中英文版本
  - 新增完整的运行指南和部署配置文档
  - 新增 FAQ 常见问题解答
  - 添加 Agent 运行效果截图展示

- **项目展示优化**
  - 更新项目标题和核心特性描述
  - 添加致谢说明，感谢原 mobile-mcp 项目
  - 更新联系方式为 QQ 交流群

### Changed

- **发版流程优化**
  - 更新发版 workflow 支持 Gitee+GitHub 双平台发布
  - 添加 DMG maker 配置

- **链接更新**
  - 更新底部链接为正确的 Gitee/GitHub 项目地址
  - 使用本地图片兼容 Gitee 平台
